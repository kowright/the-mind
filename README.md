## Get started

Configure FE .env in project root and the BE .env in `./server` by using the example .env file

Download Expo Go App

Run the backend from `\the-mind\server` using 
```
npm run dev
```

Run the frontend from `\the-mind` using

```
 npx expo start -c
```

Use QR code in frontend logs to connect to Expo Go App or click the link in the logs for the web version

Happy Playing!
